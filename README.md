# Hermes Knowledge Graph

A neon 3D WebGL visualization of Erik/Eddy/Hermes Graphify memory and knowledge graph data.

- `index.html` is the self-contained interactive graph visual.
- `preview.png` is a static screenshot preview.
- `graphify_3d.py` is the generator used to create the HTML from Graphify `graph.json`.

Generated from the local Graphify corpus at `E:/workflow/repo-learning/graphify`.

Graph size at generation time: 9,631 nodes and 16,619 connections.
